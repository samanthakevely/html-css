Colocando imagens

Para colocarmos uma imagem utilizamos a tag 'img'.

Se a imagem estiver na pasta principal (que é onde fica todos os arquivos), colocaremos ela direto.

Para por a imagem acima utilizamos: img src="eu.jpg" alt="minha foto", onde 'src' colocamos o nome do arquivo da imagem e 'alt' colocamos um texto alternativo para descrever a imagem. Para configurar o tamanho da imagem, no CSS coloquei width: 200px;


Esta imagem abaixo está salva dentro de uma pasta na nossa pasta principal chamada 'imagens'.

Para colocarmos uma imagem desta forma ESCREVA O NOME DA PASTA, EX: ‘imagens’ + ‘/’ + O NOME DO ARQUIVO SALVO NESSA SUBPASTA, 'falhanosistema.webp'

o código ficará assim: img src="imagens/falhanosistema.webp" alt="falhanosistema"
